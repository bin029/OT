# 上下班打卡记录器 PWA

一个简单的渐进式Web应用，用于记录上下班打卡时间。

## 功能特性

- 🖱️ 简单的一键打卡记录
- ⏰ 自动记录每日上班和下班时间
- 📊 智能计算每日加班时间
- 💾 本地存储，数据持久化
- 📱 PWA支持，可安装到桌面
- 🔄 离线可用

## 如何使用

1. 打开浏览器访问应用
2. **上班打卡**：点击"打卡"按钮记录上班时间
3. **下班打卡**：再次点击"打卡"按钮记录下班时间
4. 查看界面上的统计信息：
   - **本周加班**：本周累计加班时间总和
   - **本月加班**：本月累计加班时间总和
5. 查看下方的今日打卡记录：
   - 上班后只显示上班时间
   - 下班后显示上班时间、下班时间和今日加班时间
5. 支持安装为PWA应用

## 加班时间计算规则

- **工作日**：周一至周五
- **标准工作要求**：每天必须工作满9小时，上班最晚9:00打卡
- **加班开始时间**：下午18:30以后才算加班
- **计算逻辑**：
  1. **如果实际工作时长 < 9小时**：加班时间 = (下班时间 - 上班时间 - 9小时)
  2. **如果实际工作时长 ≥ 9小时**：加班时间 = max(0, 下班时间 - 18:30) - max(0, 上班时间 - 9:00)
- **特殊情况**：
  - **早退严重**：工作时长不足9小时，显示负数
  - **早退适中**：工作时长满9小时但下班早于18:30，显示0
  - **正常加班**：下班晚于18:30，显示正数
  - **迟到影响**：上班晚于9:00，会扣除相应加班时间
- **周末**：不计算加班时间（显示为0）
- **缺卡**：未完整打卡（缺少上班或下班记录）的日期，加班时间为0
