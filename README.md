# 数字时钟 PWA

一个简单的渐进式Web应用，集成了数字时钟和上下班打卡功能。

## 功能特性

- 🖱️ 简单的一键打卡记录
- ⏰ 自动记录每日上班和下班时间
- 📊 智能计算每日加班时间
- 💾 本地存储，数据持久化
- 📱 PWA支持，可安装到桌面
- 🔄 离线可用

## 如何使用

1. 打开浏览器访问应用
2. **上班打卡**：点击"打卡"按钮记录上班时间
3. **下班打卡**：再次点击"打卡"按钮记录下班时间
4. **清除记录**：点击"清除"按钮清除当日打卡记录
   - 如果已打卡2次：第一次清除下班记录，第二次清除上班记录
   - 如果只打卡1次：直接清除上班记录
5. **清除全部**：点击"清除全部"按钮清除所有历史打卡记录
   - 会显示确认对话框，防止误操作
   - 此操作不可恢复，请谨慎使用
6. **切换标签页**：使用顶部标签页切换不同视图
   - **今日记录**：显示今天的打卡详情
   - **月度详情**：显示本月每一天的加班情况和月度总计
7. 查看界面上的统计信息：
   - **本周加班**：本周累计加班时间总和
   - **本月加班**：本月累计加班时间总和
8. 支持安装为PWA应用

## 加班时间计算规则

- **工作日**：周一至周五
- **标准工作要求**：每天必须工作满9小时，上班最晚9:00打卡
- **加班开始时间**：下午18:30以后才算加班
- **混合计算逻辑**：
  - **如果下班时间 < 18:30**：加班时间 = (下班时间 - 上班时间 - 9小时)
  - **如果下班时间 ≥ 18:30**：加班时间 = max(0, 下班时间 - 18:30) - max(0, 上班时间 - 9:00)
- **特殊情况**：
  - **早退（下班 < 18:30）**：显示实际工作时长与9小时的差值（通常为负数）
  - **正常下班（18:30）**：显示0（刚好工作9小时）
  - **加班（下班 > 18:30）**：显示超出18:30的时间，减去迟到时间
  - **迟到影响**：上班晚于9:00，会相应扣除加班时间
### 🏖️ 周末（周六、周日）
- **加班计算**：下班时间 - 上班时间（任何时段都算加班）
- **注意**：如果只有一次打卡，不算加班（显示0）
- **特点**：周末全天任何工作时间都算加班，无需满足9小时要求

### 📊 通用规则
- **缺卡**：未完整打卡（缺少上班或下班记录）的日期，加班时间为0
